Step One - The very first step in creating an HTML file

Like with many aspects of coding, creating an HTML page requires the use of text editor. Go ahead and fire up your favorite. While many people in class like to use Visual Studio Code, I prefer to use the open source program Atom.

Step 2 - Really getting started

What use is a text editor if there's no text? To get us started, Atom is pretty useful. By simply typing html into the editor, the boilerplate of an html page is automatically generated. This can be particularly useful for beginers, as it shows the basic layout of an HTML page as well as the head and doctype elements.
